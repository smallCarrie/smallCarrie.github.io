目的：实现一个一直循环滚动的广告栏
分析：

  • 设置一个固定宽度的div,包裹滚动元素，超出隐藏。
  • 放两倍的广告小图片
  • 给每一个小图设定宽度
  • setInterval固定时间给第一个元素设定负left，使其左移
  • 如果负left的值等于所有广告图片（不是前面说的2倍的宽，是真实的每一个图相加的宽）的总宽度，就把left设为0。说明已经循环一遍
